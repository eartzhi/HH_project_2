### Какой кейс решаем?    
Осуществляем разведывательный анализ и очистку данных

**Метрика качества**     
В качестве критерия оценки качества предлагается рассматривать исполнение описаных в файле заданий. Параллельно давались ответы по заданиям на сайте Skillfactory.

**Что практикуем**     
Практикуем работу с библиотеками pandas, numpu и plotly.


### Краткая информация о данных
В качестве набора данных используется выгрузка резюмэ с сайта hh.ru за 2018-2019 годы.
